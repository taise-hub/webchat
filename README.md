# webchat
This is Simple WebChat using WebSocket.  
This is implemented in accordance with clean architecture, so It's easy to extend function enhancement.(just I think so...)
# DEMO 
1.
```/bin/sh
% git clone https://github.com/taise-hub/webchat.git
```
```
% cd webchat
```
```/bin/sh
% docker-compose build
```
```/bin/sh
% docker-compose up -d
```

2. 
access localhost:8080
